# RAF Event Booker Frontend

Frontend web application for the RAF Event Booker system.\
Provides a user interface for browsing events, booking tickets, and
managing user accounts.

------------------------------------------------------------------------

## Features

-  User login and registration\
-  Browse available events\
-  Book and manage reservations\
-  User profile management\
-  Client-side routing\
-  Responsive UI

## Getting Started

### Prerequisites

-   Node.js 18+\
-   npm or yarn

### Steps

1.  Clone the repository:

``` bash
git clone https://github.com/NuQrZ/RAF-Event-Booker-Frontend.git
```

2.  Install dependencies:

``` bash
npm install
```

3.  Start the development server:

``` bash
npm start
```

The app will run on:

    http://localhost:3000

------------------------------------------------------------------------
